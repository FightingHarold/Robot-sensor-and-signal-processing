# Robot-sensor-and-signal-processing
All the code for a Surveillance breadboard prototype robot, including Test codes and Final working codes

All Hardware Components Used:
1. Raspberry Pi 3 module B+
2. Raspberry Pi 2 Camera Module
3. Raspberry Pi Display
4. HDMI to HDMI Cable
5. Raspberry Pi Power Cable
6. Large Breadboard
7. Small Breadboard
8. Jumper wires-Male to Male, Male to Female, Female to Female
9. Ultrasonic sensor HC-SR04
10. Servo Motor
11. PIR Sensor
12. Buzzer
13. 3 LED’s
14. 3 –220 OHM Resistor for Led
15. Push Button
16. MCP 3008
17. Diodes
18. Fuse Holder
